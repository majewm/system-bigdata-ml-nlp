    - Apache Hadoop: 3.3.0
  • Technology stack:
    - IntelliJ IDEA Ultimate
  • Project features:
    - Spark: spark-core_2.12, spark-sql_2.12
    - build plugins: maven-compiler-plugin, maven-assembly-plugin
